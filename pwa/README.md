# SSSK PWA Frontend (Веб-інтерфейс) 📱

Цей модуль містить фронтенд-частину додатку для візуалізації графіків та взаємодії з користувачами.

## 📱 Функціональні екрани (План)
1.  **Головна** — поточний статус світла ("Світло є" / "Відключено") + таймлайн на 24 години по обраній черзі (1.1–6.2).
2.  **Статистика** — теплова карта відключень, порівняння стабільності різних черг.
3.  **Історія** — календар графіків та архів змін за день.
4.  **Пошук** — швидкий перегляд черг з найбільшим відсотком відключень.
5.  **Налаштування** — вибір своєї черги, налаштування сповіщень та темна/світла теми.

## 🔔 Сповіщення
- **Push (браузер):** Попередження за 5 чи 10 хвилин до планового відключення.
- **Telegram Bot:** Команда `/group [номер]` для отримання текстової таблиці.

## 🔐 Авторизація
- **Google Account:** Основний спосіб для збереження налаштувань.
- **Telegram Auth:** Допоміжний спосіб для прив'язки сповіщень бота.

## 🛠 Технології та Дизайн
- **Дизайн:** [🎨 SSSK Design System](./design_system.md) — Apple-style Glassmorphism.
- **Фреймворк:** Vanilla JS / Vite (планується).
- **Стилі:** CSS3 (Liquid Glass effect).
- **БД Користувачів:** Supabase (PostgreSQL).
- **Хостинг:** GitHub Pages / Netlify.
