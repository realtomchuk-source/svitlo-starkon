# SSSK PWA Frontend (Веб-інтерфейс) 📱

Цей модуль містить фронтенд-частину додатку для візуалізації графіків та взаємодії з користувачами.

## 📱 Реалізовані екрани
1.  **[Головна (index.html)](./index.html)** — поточний статус світла + 24-годинний радар відключень по обраній черзі (з інтерактивним скраббером часу).
2.  **[Графік на завтра (tomorrow.html)](./tomorrow.html)** — попередній перегляд розкладу на наступний день.
3.  **[Повний графік (full_schedule.html)](./full_schedule.html)** — деталізований таймлайн для зручного перегляду.
4.  **[Особистий кабінет (cabinet.html)](./cabinet.html)** — авторизація через Google, прив'язка Telegram, керування підписками та вибір черги.
5.  **[Архів (archive.html)](./archive.html)** — календар графіків та архів змін за минулі дні.
6.  **[Спільнота (community.html)](./community.html)** — модуль залучення та зворотного зв'язку користувачів.
7.  **[Адмін-панель (admin.html)](./admin.html)** — закритий інтерфейс для модерації, ізольований від основного дизайну.

## 🔔 Сповіщення
- **Push (браузер):** Попередження за 5 чи 10 хвилин до планового відключення.
- **Telegram Bot:** Команда `/group [номер]` для отримання текстової таблиці.

## 🔐 Авторизація
- **Google Account:** Основний спосіб для збереження налаштувань.
- **Telegram Auth:** Допоміжний спосіб для прив'язки сповіщень бота.

## 🛠 Технології та Архітектура
- **Дизайн:** [🎨 SSSK Design System](./design_system.md) — Apple-style Glassmorphism.
- **Фреймворк:** Vanilla JS (ES Modules).
- **Стилі:** CSS3 (Liquid Glass effect, CSS Variables).
- **Бекенд та Auth:** Supabase (PostgreSQL, Google Auth).
- **Хостинг:** GitHub Pages / Netlify.
