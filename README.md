# Svitlo Starkon (Світло-Старкон) 💡

**PWA-додаток для оперативного моніторингу графіків відключень електроенергії** для Старокостянтинівської територіальної громади.

---

## 🚀 Про проєкт
Svitlo Starkon — це відкрита екосистема, яка перетворює складні PNG-графіки обленерго на зручні цифрові дані. Проєкт побудований за принципом **Serverless Monorepo** на базі GitHub Actions.

### 🏘️ Географія покриття
- **Основне:** місто Старокостянтинів (черги 1.1–6.2)
- **Розширення:** Старостинські округи та територіальна громада.

---

## 🏗️ Архітектура (Monorepo)
Проєкт розділений на два ключові модулі:

1.  **[⚙️ Parser](./parser/)** — Python-модуль, що сканує сайт обленерго та Telegram, розпізнає (OCR) графіки та зберігає їх у JSON.
2.  **[📱 PWA Frontend](./pwa/)** — Веб-додаток (GitHub Pages), який візуалізує таймлайни та надсилає сповіщення.

**Потік даних:**
`hoe.com.ua (PNG)` → `GitHub Actions` → `parser/data/current.json` ← `PWA (GitHub Pages)`

---

## 📊 Основний функціонал (у розробці)
- ✅ **Таймлайн 24 год:** візуалізація відключень по обраній черзі.
- ✅ **Dual-Source моніторинг:** сайт + Telegram для максимальної точності.
- 🔜 **Push-сповіщення:** "Відключення за 10 хвилин".
- 🔜 **Аналітика:** теплова карта відключень та порівняння черг.
- 🔜 **Авторизація:** збереження своєї черги через Google/Telegram.

---

## 🔧 Технологічний стек
- **Backend/Parser:** Python 3.10, Tesseract OCR.
- **Frontend:** PWA (Vanilla JS / CSS3), GitHub Pages.
- **Database:** Flat JSON (сховище графіків) + Supabase (дані користувачів).
- **Automation:** GitHub Actions.
