# 🌟 Світло-Старкон — коли буде світло у Старокостянтинові?

**Світло-Старкон** — це безкоштовний PWA-додаток для оперативного моніторингу графіків відключень та включень електроенергії для Старокостянтинівської територіальної громади.

---

## 📱 Для кого та навіщо
Мешканці Старокостянтинівської громади, які хочуть:
- ✅ **Знати точно**: коли буде світло, коли відключать у своїй черзі (1.1–6.2).
- ✅ **Отримувати сповіщення**: "За 15 хв відключать" або "Світло з'явиться".
- ✅ **Бачити історію**: всі графіки за місяць + зміни ("сьогодні змінили 3 рази").
- ✅ **Аналізувати статистику**: яка черга страждає більше, порівняння сьогодні/вчора.

---

## �️ Географія покриття
- **Основне:** місто Старокостянтинів — черги 1.1–6.2 (усі мікрорайони).
- **Розширення:** Старостинські округи та територіальна громада (окремі черги громад).
- **Перспектива:** Реплікація проєкту для сусідніх громад області.

---

## �🏗️ Архітектура проєкту (Monorepo)
Проєкт побудований за принципом **Serverless Architecture** для максимальної надійності та нульової вартості утримування:

1.  **[⚙️ Parser](./parser/)** — Python-ядро (Tesseract OCR), що працює на GitHub Actions.
2.  **[📱 PWA Frontend](./pwa/)** — Сучасний інтерфейс на базі GitHub Pages / Netlify.
3.  **[📡 API/Data](./parser/data/)** — Структуровані JSON-дані.
4.  **[🗄️ Database]** — Supabase (PostgreSQL для підписок користувачів).

**Потік даних:**
`hoe.com.ua PNG` → `GitHub Actions` → `parser/data/unified_schedules.json` ← `PWA`

## 🚀 Розгортання та Масштабування

**Локальний запуск:**
```bash
# Тест парсера
python parser/src/parse_schedule.py

# Тест PWA (локально)
cd pwa && npx serve .
```

**Масштабування:**
- **До 1000 к-чів:** GitHub Free / Netlify ✅
- **10k+ користувачів:** Автоматичний деплой через Netlify + Supabase Backend ✅
