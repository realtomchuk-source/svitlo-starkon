# Svitlo Starkon (Світло-Старкон) 💡

Система моніторингу та реконструкції графіків відключень електроенергії для мешканців Старокостянтинівської громади.

## 🚀 Про проєкт
Це повністю **безсерверна (Serverless)** система, яка використовує GitHub Actions як обчислювальну потужність та GitHub Pages як платформу для веб-застосунку.

### Ключові особливості:
- **Піксельний парсинг:** Скрипти на Python аналізують офіційні PNG-графіки "Хмельницькобленерго", перетворюючи їх на числові дані з точністю до хвилини.
- **Модель істини:** Система зберігає всі версії графіків за добу, реконструюючи "офіційний план" та вираховуючи реальний відсоток відключень.
- **Індекс нестабільності:** Аналіз того, наскільки сильно змінювалися плани протягом дня.
- **PWA:** Легкий веб-додаток, який працює офлайн та може бути встановлений на телефон.

## 🏗 Архітектура даних
Система не використовує традиційну базу даних. Замість цього всі дані зберігаються у форматі JSON у папці `/data`:
- `current.json`: Актуальний графік на поточну добу.
- `tomorrow.json`: Графік на наступну добу (з'являється після ~20:00).
- `history/`: Архів графіків за конкретні дні.
- `versions/`: Хронологія змін графіка протягом однієї доби.

## 🛠 Технологічний стек
- **Backend/Parser:** Python (PIL, Requests)
- **CI/CD:** GitHub Actions
- **Database:** Flat JSON files
- **Frontend:** PWA (HTML5, Vanilla JS, CSS3)
- **Deployment:** GitHub Pages
